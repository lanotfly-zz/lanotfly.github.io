#### SQL grammar order and execution order is different

一般而言

#### SQL grammar order：
~~~~
SELECT[DISTINCT]
FROM
WHERE
GROUP BY
HAVING
UNION
ORDER BY
~~~~
#### execution order：
~~~~
FROM
WHERE
GROUP BY
HAVING
SELECT
DISTINCT
UNION
ORDER BY
~~~~
